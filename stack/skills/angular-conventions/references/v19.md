# Angular v19

Released 2024-11-19. **End-of-life as of ~2026-05** - it is in the working range but no longer receives fixes; treat moving off it as debt reduction. The rest of SKILL.md applies; this file states what v19 does NOT yet have that the skill's forward idioms assume, plus its v19-era API spellings. Load it only if the workspace is on v19. Fact-checked against angular.dev.

## Runtime

- Node.js `^18.19.1 || ^20.11.1 || ^22.0.0`, TypeScript `>=5.5.0 <5.9.0` (19.2.x). RxJS `^6.5.3 || ^7.4.0`.
- Support: **end-of-life** - no active or LTS patches.

## New in v19 - the baseline the skill body assumes starts here

- **`standalone: true` is the implicit default** - no longer written; `NgModule` components opt out with `standalone: false`.
- Signal inputs/outputs are stable: `input()`, `input.required<T>()`, `output()`, `model()` (all reached stable in v19.0).
- `@let` template variables are stable.
- `afterRenderEffect` is available.

## Experimental in v19 - usable but expect churn, version-tag every snippet

- `linkedSignal` ships experimental (stabilizes v20).
- `resource` and `rxResource` ship experimental in v19.0; `httpResource` ships experimental in v19.2. The whole family stays experimental until v22 - pin to the installed version.
- Note the pre-v20 spellings: `resource` uses `request` (renamed `params` in v20) and `rxResource` uses `loader` (renamed `stream` in v20). `rxResource` only becomes always-streaming at v19.2.
- Incremental hydration is developer preview (stabilizes v20).

## NOT yet available in v19 - do not reach for these

- Zoneless is not stable here (still experimental) - keep `zone.js`.
- **Signal Forms does not exist yet** (ships experimental in v21) - use typed reactive forms (`FormGroup<T>`).
- `@angular/aria` does not exist yet (ships developer preview in v21).
- Material's `matButton` attribute and the `--mat-*` token rename are v20 - v19 uses the `mat-button` family and `--mdc-*` tokens.
